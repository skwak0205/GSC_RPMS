This is the readme file for autoupgrade.

Use the following steps to setup autoupgrade of IEC:

1. Create/update  IEFVersionDetails.xml and place it in <application folder>\integrations\autoupgrade folder.
3. Create another folder "installer" in <application folder>\integrations\autoupgrade folder.
4. Copy the IEF client installable zip file to <application folder>\integrations\autoupgrade\installer folder. 

    Note: The IEF client installable zip file should not be upzipped and should be copied to the installer folder in     the zipped form.
