Configuration of DMCTool

0. Introduction

This help sheet describes how to configure DMCTool before starting.

Click to restart
 
 
1. General parameters

Select "Tools/Options..." command and "General" tab page.

In the prefix field, enter a 3 letters prefix (default is CUS). This prefix is used for naming rules for new customization names and new types. The prefix is specific for each company.

In the Working folder field, enter or select a folder path, where new customizations and all the related files are going to be created.

Click to complete
 
 
2. Configurations of MatrixOne

Select "Tools/Options..." command and "ENOVIA VPM Server Configuration" tabpage.

Enter or select the Matrix Rmi path folder (ENOVIA Collaboration Platform Server) for your local configuration.

Enter or select the Matrix path (ENOVIA Collaboration Platform ModelingStudio) folder for this new configuration.

Enter or select the jdk path for your Enovia VPM server

Click Ok or Apply to confirm parameters for this new configuration.

Click to complete
 
 
3. External API

Select "Tools/Options..." command and "External API" tab page.

Some additional and optional functionalities are available if some external addins are downloaded from the web. Then the path for the downloaded product must be specified in this panel. Note that DMCTool must be restarted if one of this path is changed.

Path of JDIC API: the Java Desktop Integration Components is necessary for accessing to the "Help/How to..." facility.

Click to complete
