Create a new customization

0. Introduction

This help sheet describes how to create a new customization for an existing modeler, and what are the several steps to follow with DMCTool in order to deploy it on a test PLM database.

Click to restart
 
 
1. Create a new Customization

Select File/New customization... Select the Modeler that you want to customize from the “Modeler to customize” list.

Enter a name for your new customization. The customization Name must start by a 3 letters prefix (default is CUS). This prefix can be modified through the Tools/Options panel (General tab page).

Then click Ok button to create new customization.

Click to complete
 
 
2. Create a new Type for your customization

You are now on the Customization tabpage

Select your customization name on the left side, and create a new type using the “Insert type…” command from the contextual menu or from the “Insert/ Type…” menu.

Select the “Type to derive” from the list and enter a name for the new type. The type Name must start by the 3 letters prefix (this prefix can be customized through the Tools/Options panel (General tabpage).

Then click Ok to create type.

Click to complete
 
 
3. Define new attributes for the new type

Select the new created type in the tree (in the left side) and create a new attribute using the “Insert Attributes…” command from the contextual menu or from the “Insert/ Attribute…” menu.

Enter a name for the new attribute, select the type from the list, and enter all necessary properties requested for your attribute (Protection flag, Default Value, Max length, etc…)

Then click Create button.

Perform this step for all other attributes if necessary.

Click to complete
 
 
4. Define IdentierSet for the new type

If an identifier set is requested for the type, an empty Identifier Set appears in the tree with a red cross. If no such identifier set appears, then you can click to complete and skip this step.

Otherwise, select it in the tree side. In the “Define Identifier set”panel, enter the Name for the new identifier set, and select attributes that you want to take part for the identifier. Note that only attributes with Protection flag=Free are available in the list.

Click Ok button to validate the identifier set creation.

Then, the empty identifier set with the red cross is replaced by the created identifier set.

Click to complete
 
 
5. Repeat steps 2 to 4 for all types to be created in your customization
Click to complete
 
 
6. Save the new customization

Once you have created all necessary types and defined all attributes, you can save your work using “File/Save/Customization…”.

The customization is then automatically saved under the Working folder specified in the Tools/Options (General tabpage) parameters. The metadata filename is automatically generated with the customization name.

Click to complete
 
 
You have successfully finished this tutorial.
